ECOVACS automation

This repository contains a Python script to control and monitor an ECOVACS robot via uiautomator2 and MQTT. Features:

- UI automation using `uiautomator2`.
- Home Assistant MQTT discovery for rooms and controls.
- Map screenshot capture and upload to Home Assistant.

Notes:
- Adjust `ecovacs.py` configuration for your MQTT broker and device address.
- Do not commit sensitive credentials.
